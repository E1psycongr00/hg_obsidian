## 코드

### Code

```python
import sympy as sp
sp.init_printing()


# 기호 정의
V1, r1, r2, x1, x2, s = sp.symbols(r"V1, r_1, r_2, x_1, x_2, s")

# 식 정의하기
Z = sp.sqrt((r1 + r2/s)**2 + (x1 + x2)**2)
I_2 = V1 / Z
P = I_2**2 * (1/s - 1) * r2

# 극값 구하기
dP_ds = sp.diff(P, s)
s_p = sp.solve(dP_ds, s)


```
