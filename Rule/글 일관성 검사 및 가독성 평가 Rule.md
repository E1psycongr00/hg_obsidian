
# 📑 글 일관성 검사 및 가독성 평가 Rule

---

## 📌 일관성 검사 (Consistency Check)

### Rule 1: 문단 간 주요 키워드의 일관성 유지
- **설명**: 글의 각 문단이 같은 주제를 일관되게 다루어야 합니다.
- **예시**:
    - 👍 첫 문단에서 '기후 변화'를 다루었다면, 이후 문단에서도 같은 주제 중심으로 작성
    - 👎 '기후 변화'에서 '사회적 문제'로 갑작스럽게 전환

### Rule 2: 불필요한 주제 전환 방지
- **설명**: 주제를 벗어난 내용이 포함되지 않도록 제한합니다.
- **예시**:
    - 👍 'AI 기술'을 주제로 삼았다면, '문화적 영향'으로 전환하지 않음
    - 👎 'AI 기술'에서 '영화 산업'으로 갑작스러운 전환

### Rule 3: 핵심 메시지 명확성
- **설명**: 각 문단이 중심 주제를 뒷받침해야 합니다.
- **예시**:
    - 👍 서론-배경, 본론-핵심 내용, 결론-요약 및 제안
    - 👎 논리적 일관성이 없는 문단 배치

## 📌 가독성 계산 (Readability Score)

### Rule 1: 문장 길이 제한 (20~25단어 이하 권장)
- **설명**: 문장이 너무 길어지면 읽기 어려워질 수 있습니다.
- **예시**:
    - 👍 "기술 혁신은 복잡한 과정입니다."
    - 👎 "기술 혁신은 복잡하고, 많은 요소들이 얽혀 있는 매우 어려운 과정입니다."

### Rule 2: 단어 다양성 점수
- **설명**: 동일한 단어를 반복하지 않고 다양한 어휘를 사용해야 합니다.
- **예시**:
    - 👍 '중요하다' -> '핵심적이다', '필수적이다'
    - 👎 '중요하다'만 반복

### Rule 3: 단락 길이 제한 (3~5문장 권장)
- **설명**: 한 문단이 너무 길거나 짧지 않도록 합니다.
- **예시**:
    - 👍 3~5문장으로 구성된 단락
    - 👎 1문장만으로 구성된 단락

## 📌 추가 평가 요소

### Rule 1: 수동태와 명사구 지양
- **설명**: 능동형 문장을 우선 사용합니다.
- **예시**:
    - 👍 "기술은 빠르게 발전하고 있다"
    - 👎 "기술의 빠른 발전이 관찰되고 있다"

### Rule 2: 명확하고 간결한 표현
- **설명**: 불필요한 형용사와 부사를 제거합니다.
- **예시**:
    - 👍 "중요한 사실"
    - 👎 "매우 중요한 핵심 사실"
