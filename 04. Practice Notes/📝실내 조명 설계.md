# 📝실내 조명 설계

## 1. 실지수(Room Index) 구하는 방법

### 실지수란?
- 실내의 형상을 나타내는 무차원 수치
- 조명 설계 시 광속법 계산에 필수적인 요소
- 실의 크기와 등기구 높이의 관계를 나타냄

### 실지수 계산 공식
```
실지수(K) = (L × W) / [H × (L + W)]
```

**여기서:**
- L: 실의 길이 (m)
- W: 실의 폭 (m)  
- H: 작업면에서 등기구까지의 높이 (m)

### 실지수 범위와 의미
- K < 1: 좁고 높은 실 (복도, 계단실 등)
- 1 ≤ K < 3: 일반적인 실 (사무실, 교실 등)
- K ≥ 3: 넓고 낮은 실 (공장, 체육관 등)

## 2. FUN = ADE 공식

### 공식의 의미
**F × U × N = A × D × E**

**각 요소 설명:**
- **F**: 등기구 1개당 광속 (lm)
- **U**: 이용률 (Utilization Factor)
- **N**: 등기구 개수
- **A**: 실의 면적 (m²)
- **D**: 요구 조도 (lx)
- **E**: 보수율 (Maintenance Factor)

### 공식 활용법

#### 1) 필요한 등기구 개수 구하기
```
N = (A × D × E) / (F × U)
```

#### 2) 평균 조도 구하기
```
D = (F × U × N) / (A × E)
```

#### 3) 필요한 광속 구하기
```
F = (A × D × E) / (U × N)
```

### 각 계수 구하는 방법

#### 이용률(U) 구하기
1. 실지수(K) 계산
2. 천장 반사율, 벽면 반사율, 작업면 반사율 확인
3. 등기구 배광 특성 확인
4. 이용률표에서 해당 값 찾기

#### 보수율(E) 설정
- 청정한 환경: 0.8~0.9
- 보통 환경: 0.7~0.8
- 더러운 환경: 0.6~0.7

## 3. 등기구 관련 설명

### 등기구의 분류

#### 1) 배광 특성에 따른 분류
- **직접조명형**: 광속의 90% 이상이 하향 조사
- **반직접조명형**: 광속의 60~90%가 하향 조사
- **전반확산조명형**: 상하 광속이 40~60% 범위
- **반간접조명형**: 광속의 60~90%가 상향 조사
- **간접조명형**: 광속의 90% 이상이 상향 조사

#### 2) 설치 방법에 따른 분류
- **매입형**: 천장에 매입 설치
- **노출형**: 천장 표면에 직접 설치
- **펜던트형**: 천장에서 매달아 설치
- **스탠드형**: 바닥에 설치

### 등기구 선정 기준

#### 1) 기능적 측면
- 요구 조도 만족 여부
- 균제도 확보 가능성
- 글레어 방지 성능
- 에너지 효율성

#### 2) 경제적 측면
- 초기 설치비용
- 운영 및 유지비용
- 램프 교체 주기
- 전력 소비량

#### 3) 미적 측면
- 실내 디자인과의 조화
- 등기구 외관
- 빛의 질감과 색온도

### 등기구 배치 방법

#### 1) 균등 배치
- 등기구를 일정한 간격으로 배치
- 일반 사무실, 교실에 적합
- 계산이 간단하고 시공이 용이

#### 2) 국부 조명
- 특정 작업 영역만 집중 조명
- 정밀 작업장, 독서실에 적합
- 에너지 절약 효과

#### 3) 혼합 조명
- 전반조명 + 국부조명 조합
- 사무실, 작업장에 많이 사용
- 작업 효율성과 쾌적성 향상

## 4. 실제 계산 예제

### 예제: 사무실 조명 설계
**조건:**
- 실 크기: 12m × 8m
- 천장 높이: 3m, 작업면 높이: 0.8m
- 요구 조도: 500 lx
- 형광등 40W × 2등용 (3200 lm)

**계산 과정:**

1) **실지수 계산**
   ```
   H = 3 - 0.8 = 2.2m
   K = (12 × 8) / [2.2 × (12 + 8)] = 96 / 44 = 2.18
   ```

2) **이용률 및 보수율 설정**
   - 이용률(U): 0.6 (이용률표 참조)
   - 보수율(E): 0.8 (일반 사무실)

3) **필요 등기구 개수 계산**
   ```
   N = (A × D × E) / (F × U)
   N = (96 × 500 × 0.8) / (3200 × 0.6)
   N = 38,400 / 1,920 = 20개
   ```

## 5. 주의사항 및 팁

### 계산 시 주의사항
- 실지수 계산 시 작업면 높이 고려 필수
- 이용률표는 등기구 제조사별로 다름
- 보수율은 환경 조건을 정확히 파악하여 설정
- 글레어 방지를 위한 등기구 배치 고려

### 실무 팁
- 계산 결과는 이론값이므로 10~20% 여유 고려
- 스위치 회로 구성을 고려한 등기구 배치
- 비상조명, 유도등 설치 기준 동시 고려
- LED 등기구 사용 시 광속 감소율 고려
