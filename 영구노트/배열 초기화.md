## 내용(Content)

Java의 배열을 처리하기 위해서는 미리 공간을 정해 놓고 사용해야 한다.

```java
int[] arr = new int[5];
for (int i = 0; i < 5; i++) {
	arr[i] = i;
}
```

for문 없이 생성과 동시에 초기화도 가능하다.
```java
int[] arr = new int[] {1,2,3,4,5};
```
