### 객체 분해 할당
>[!summary] 객체 분해 할당
>현대 언어에서 지원하는 특별한 문법 중 하나이며, 객체의 속성을 해체하여 변수에 할당하는 것을 의미한다. 

객체 분해 할당의 장점은 비교해보면 쉽게 알 수 있다.

```js
function Person(name, age, job) {
    this.name = name;
    this.age = age;
    this.job = job;
}

const person = new Person('John', 30, 'Software Engineer');

// 객체 분해 할당 x
```
