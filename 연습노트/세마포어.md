### 세마포어란
>[!summary] 세마포어
>하나 또는 여러 프로세스/스레드가 critical section에 접근 가능하도록 하도록 하는 시그널 메커니즘이다.

세마포어의 개념을 수도 코드를 통해 살펴보자

```text
class Semaphore {
	int value = 5;
	int guard = 0;
}

Semaphore::wait() {
	while (test_and_set(&guard)) {
		if (value == 0) {
			현재 스레드를 큐에 넣음;
			guard = 0;
			go to sleep;
		} else {
			value -= 1;
			guard = 0;
		}
	}
}

Semaphore::signal() {
	while(test_and_set(&guard)) {
		if (큐에 하나라도 대기중) {
			하나를 깨워서 준비시킨다 in 큐;
		} else {
			value += 1;
		}
		guard = 0;
	}

}
```

- 바이너리 세마포어
	- value = 1
	- 상호 배제 조건을 만들어줌(Mutex와 유사)
- 카운팅 세마포어
	- value > 1
	- 여러 프로세스/쓰레드가 해당 자원에 접근 가능
