## 원문

### 원리
2D 누적합의 공식을 활용한다.

$$ S(x2, y2) = S(x2, y1-1) + S(x1-1, y2) - S(x1-1, y1-1) + a(x1-1, y1-1) $$
위 식에서 영감을 얻어서 2차원 쿼리 업데이트를 할 수 있다.

예를 들어 (x1, y1) -> (x2, y2) 까지 4각형 구간을 +1씩 업데이트 한다고 가정하자

그러면 우리가 업데이트 해야 할 값은 4개이다
1.  S(x1, y1)++
2. S(x2+1, y1)--
3. (x1, y2+1)--
4. (x2+1, y2+1)++
